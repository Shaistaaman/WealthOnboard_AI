1.  Project Overview

Client: WealthOnboard AI

Project Goal: To create a comprehensive, user-friendly, and secure website that provides both front-end and back-office portal features for interactive broker services.

Target Audience: Individual and corporate clients seeking online brokerage services, financial advisors, and bank administrators.

Website Tone: Professional, trustworthy, modern, and user-centric.

Project Summary
WealthOnboard AI MVP - A comprehensive fintech platform for interactive broker services with both client-facing and back-office functionality, built on AWS serverless architecture.

Key Architecture Approach:
No IaC/CDK: Manual AWS service setup
Direct API Calls: Frontend directly calls AWS services via API Gateway
Backend Structure: Lambda functions organized in a /backend folder
Serverless Stack: AWS Lambda, API Gateway, DynamoDB, Step Functions, Bedrock
